# **SSD数理モデル：統一用語辞書 v2.4 (Bias & Inertia Update)**

本辞書は、構造主観力学（SSD）を「非平衡熱力学の主観的記述」として定義し、AI、人間、動物、機械を含むあらゆる知性の統合シミュレーションを可能にするものである。

## **0\. 基礎概念の物理統合 (Core Physics)**

**「世界は全周波数で鳴っているが、私は私の受信機でしか聴けない」**

| SSD概念 (Symbol) | 物理学的対応 | 新・定義・実装仕様 |
| :---- | :---- | :---- |
| **素流圧** (Elementary Flux Pressure: $x$) | **物理刺激 / 入力ベクトル** (Raw Stimulus) | **【フィルタリングされたエネルギー流】** 潜在的物理状態 $S(t)$ から、エージェントの**受容体（センサー）が切り取った**部分集合。 例：人間にとっての $x$ には紫外線は含まれないが、モンシロチョウにとっての $x$ には紫外線が含まれる。 $$x = \text{Sensor}(S(t)) \in \mathbb{R}^n$$ |
| **構造** (Structure: $A$) | **変換行列 / ドライバ** (Transformation Matrix) | **【意味の生成 兼 信号取得装置】** ハードウェア（肉体・センサー）とソフトウェア（解釈・記憶）の統合体。 **スペクトル帯域**（赤外線が見えるか？）と**解像度**（1℃の違いが分かるか？）を定義する。 センサーが捉えられない信号は、世界に存在しても $A$ にとっては「無」である。 $$y = A_{logic} \cdot x$$ |
| **内部熱** (Internal Heat: $E$) | **摩擦熱 / 損失エネルギー** (Friction / Loss) | **【構造的摩擦】** 素流圧 $x$ が構造 $A$ を通過する際に、スムーズに処理（整合）できずに発生する内部抵抗。 高解像度のセンサーを持つ個体は、ノイズも大量に拾うため、必然的に熱（ストレス）が発生しやすい。 $$E \approx \|x - A_{ideal}y\|$$ |

**「計算可能な神（Server-Side Logic）」**

| 変数名 | 数理記号 | 定義・実装仕様 |
| :---- | :---- | :---- |
| **物理行列** (Universal Matrix) | $A_{phys}$ | **【世界の法則（API仕様書）】** 全エージェントに共通する、変更不能な環境の遷移ルール。 重力定数や光速度などが記述される不変の領域。 実装上は static な環境クラス。 |
