# **数理モデル改訂案：物理層と神経制御層の分離 (Bio-Physical Separation Model)**

## **1\. 改訂の目的**

従来のSSD数理モデルでは、「探索温度 ($T$)」や「感受性 ($c_1$)」といったパラメータを単一の数式内で扱っていた。本改訂案では、これを以下の二層に明確に分離することで、モデルの普遍性と拡張性を飛躍的に向上させることを目的とする。

1. **物理層（The Physics Layer）**: 熱力学・統計力学に基づく、普遍的でシンプルな状態遷移エンジン。  
2. **制御層（The Control Layer）**: 神経物質相互作用に基づく、個体固有のパラメータ調整モジュール。

## **2\. 第1層：物理層（ユニバーサル・エンジン）**

この層には「性格」や「感情」は存在しない。存在するのはエネルギー保存則と確率論だけである。このエンジンは人間だけでなく、組織、経済、物理現象にもそのまま適用可能である。

### **2.1. エネルギー保存則（熱の蓄積）**

処理しきれなかった仕事（残差）は、必ず内部エネルギー（熱）として保存される。

$$
\frac{dE}{dt} = (\text{入力仕事}) - (\text{整合仕事}) - (\text{自然放熱})
$$

$$
\frac{dE}{dt} = \alpha_{\text{in}} \cdot \|p\| - \|j\| - \beta \cdot E
$$

* $E$: 未処理圧（熱）。システムの不安定化エネルギー。  
* $p$: 入力意味圧。  
* $j$: 整合流（出力）。  
* $\alpha_{\text{in}}$: 入力効率（物理的な受容能力）。  
* $\beta$: 放熱係数（物理的な減衰）。

### **2.2. 相転移の法則（探索温度による確率的跳躍）**

構造が変化（跳躍）する確率は、障壁の高さと、その場の「温度」のみによって決定される（アレニウスの式的アプローチ）。

$$P(\text{Leap}) \propto \exp\left( -\frac{\Delta F}{T} \right)$$

* $P(\text{Leap})$: 跳躍発生確率。  
* $\Delta F$: 跳躍に必要なエネルギー障壁（構造の硬さ）。  
* $T$: **探索温度**。分子（選択肢）の運動エネルギー。

## **3\. 第2層：制御層（ニューロ・チューナー）**

「心」「性格」「病理」はすべてこの層に記述される。神経物質は物理層のパラメータを動的に書き換える「変調器（Modulator）」として機能する。

### **3.1. 探索温度 ($T$) の決定**

探索温度は、物理的な熱 ($E$) を、神経物質のバランスによって増幅・抑制した結果である。

$$T(t) = \frac{\text{Dopamine}(t)}{\text{Serotonin}(t) + \epsilon} \cdot E(t)$$

* **ドーパミン（駆動）**: 熱 ($E$) を運動エネルギー ($T$) に変換するアクセル。  
  * *増加要因*: 報酬予測、新規性、ノルアドレナリン刺激。  
* **セロトニン（抑制）**: 温度上昇を抑え、構造を安定させるブレーキ。  
  * *減少要因*: 過度な熱 ($E$) の持続（ストレス損傷）。

### **3.2. 入力感度 ($\alpha_{\text{in}}$) の決定**

外部からの意味圧をどれだけ強く取り込むかは、警戒システムによって制御される。

$$\alpha_{\text{in}}(t) = \text{BaseSensitivity} \times \text{NorAdrenaline}(t)$$

* **ノルアドレナリン（増幅）**: 危機検知により分泌され、入力ゲインを引き上げる。  
  * *結果*: わずかな意味圧でも、物理層には巨大な入力として伝達され、急激な $E$ の蓄積を招く。

### **3.3. 障壁認識と緩衝 ($\Delta F, \beta$)**

苦痛（不整合）の感じ方や、熱の逃がし方も物質に依存する。

* **エンドルフィン（緩衝）**: 物理層の放熱係数 $\beta$ を一時的に増大させる、あるいは障壁 $\Delta F$ を低く見積もらせる（鎮痛）。

## **4\. このモデルで説明できる現象（病理と性格）**

物理層（エンジン）は正常でも、制御層（チューニング）次第で挙動は劇的に変化する。

### **ケースA：短気・敏感な性格**

* **設定**: ノルアドレナリンのベース分泌量が多い。  
* **現象**: 物理層への入力ゲイン $\alpha_{\text{in}}$ が高いため、些細な意味圧 $p$ でも急速に熱 $E$ が溜まり、即座に跳躍（怒り・反応）する。

### **ケースB：うつ状態（学習性無力感）**

* **設定**: セロトニン枯渇、かつドーパミン不全。  
* **現象**: 熱 $E$（ストレス）は大量に蓄積されているにもかかわらず、温度 $T$ が上昇しない。  
  * $T$ が低いため、相転移の法則 $P \propto \exp(-\Delta F/T)$ により跳躍確率がゼロに近づく。  
  * 結果、苦痛な現状から「逃げる（跳躍する）」ことが物理的に不可能になり、スタックする。

### **ケースC：躁状態**

* **設定**: ドーパミン過剰分泌。  
* **現象**: 熱 $E$（ストレス根源）が存在しなくても、分子がドーパミン駆動で激しく運動し、温度 $T$ が異常上昇する。  
  * 障壁 $\Delta F$ を容易に飛び越え、無秩序な跳躍を繰り返す。

## **5\. 実装への示唆 (Python擬似コード)**

class BioPhysicalAgent:  
    def __init__(self):  
        # Layer 1: Physics Engine (Stateless laws)  
        self.heat_E = 0.0  
        self.barrier_F = 1.0  
          
        # Layer 2: Neuro Controller (Stateful parameters)  
        self.dopamine = 0.5  
        self.serotonin = 0.5  
        self.noradrenaline = 0.5  
      
    def update(self, pressure_p, dt):  
        # --- Layer 2: Control & Modulation ---  
        # 感度調整（ノルアドレナリン）  
        input_gain = 1.0 + 2.0 * self.noradrenaline  
          
        # 温度決定（ドーパミン / セロトニン）  
        # 熱Eがあるほど温度は上がるが、その係数は物質が決める  
        sensitivity_c1 = self.dopamine / (self.serotonin + 1e-6)  
        temperature_T = sensitivity_c1 * self.heat_E + 0.01  
          
        # --- Layer 1: Physics Calculation ---  
        # エネルギー保存則: 熱の更新  
        effective_input = input_gain * abs(pressure_p)  
        dissipation = 0.1 * self.heat_E  
        self.heat_E += (effective_input - dissipation) * dt  
          
        # 相転移の法則: 跳躍判定  
        leap_probability = math.exp(-self.barrier_F / temperature_T)  
          
        if random.random() < leap_probability:  
            self.leap()

## **6\. 結論**

この分離モデルにより、SSDは以下の利点を得る。

1. **計算の軽量化**: 物理層は単純な四則演算のみで動作し、大規模シミュレーションが可能。  
2. **説明性の向上**: 「なぜその行動をとったか」を、「エネルギー的要因（物理）」と「気質的要因（制御）」に分けて説明できる。  
3. **汎用性**: 制御層を「資金」や「情報」に置き換えれば、そのまま経済・組織モデルへ転用可能。
